FI VS Code Server

Instructions

  1. Follow the instructions in the console below to grant access to your server.
  2. Connect to your server in one of two ways:
    • Option A: follow the instructions in the console and open the VS Code client in your browser
    • Option B: open your local VS Code client on your personal computer and connect to the running server (F1 -> "Remote-Tunnels: Connect to Tunnel...")
More details on these options are given on the FI Wiki VS Code page.
